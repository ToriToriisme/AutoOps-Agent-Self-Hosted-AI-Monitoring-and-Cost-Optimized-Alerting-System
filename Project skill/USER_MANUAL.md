# Sổ tay Người dùng / Quản trị viên

1. **Cách xem Dashboard:** Đăng nhập `http://[IP]:3000`, quan sát các Node màu Xanh (Ổn), Vàng (Có lỗi Small/Medium), Đỏ (Critical).
2. **Quy trình Phê duyệt (Approve):**
   - Vào tab *Pending Approvals*.
   - Đọc lý do AI đề xuất.
   - Bấm nút `Approve` để cấp phép cho AI chạy, hoặc `Reject` để hủy bỏ.
