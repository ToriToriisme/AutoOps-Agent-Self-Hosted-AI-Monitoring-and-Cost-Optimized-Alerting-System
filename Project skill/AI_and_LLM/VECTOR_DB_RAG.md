# Thiết lập Vector DB & RAG

RAG giúp AI Agent không bị ảo giác bằng cách tham chiếu lịch sử [20].
- **Database:** Sử dụng Milvus hoặc Qdrant.
- **Dữ liệu Vector hóa:**
  1. *Playbooks:* Các quy trình xử lý chuẩn của công ty (SOP).
  2. *Lịch sử lỗi:* "Lỗi X từng xảy ra lúc Y, đã được giải quyết bằng Tool Z".
